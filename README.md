# Travel Bud - Bus Booking System

## Overview
Travel Bud is a web-based bus booking system built with **ASP.NET MVC, SQL Server, HTML, and CSS**.  
It allows users to search for bus routes, view details, and confirm bookings.  
Admins can manage routes, view all bookings, and perform CRUD operations.

---

## Features
- 🚌 **Route Management**: Add, edit, and delete bus routes.
- 👥 **Booking Management**: View all bookings, search by passenger/destination/date, and edit/delete bookings.
- 🔍 **Search Functionality**: Users can search routes by start location, destination, and travel date.
- 🎨 **Frontend Design**: Responsive homepage with hero image, booking form, styled tables, and booking confirmation.
- 💾 **Database Integration**: SQL Server with identity columns for buses and routes.

---

## Tech Stack
- **Backend**: ASP.NET MVC (C#)
- **Database**: SQL Server
- **Frontend**: HTML, CSS, Bootstrap
- **ORM**: Entity Framework Core

---

## Setup Instructions
1. Clone the repository.
2. Configure your SQL Server connection string in `appsettings.json`.
3. Run migrations to create the database schema.
4. Launch the project in Visual Studio or `dotnet run`.
